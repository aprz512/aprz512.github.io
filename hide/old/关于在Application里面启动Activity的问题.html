后来我想起来 activity 的启动流程，点击 launcher 的图标其实也是调用了 startActivity，所以说是先启动的 MainActivity，然后再启动的 SecondActivity。activity 启动流程中有这样的逻辑判断，如果在启动一个新的 activity 的过程中，有别的 activity 正在启动，会先等待这个 activity 启动完成，不然的话，生命周期就乱了。

其实最初我以为在 Application 里面是不能启动 activity 的，因为 Application 没有窗口 token。然后自己跟踪了一下源码，token 为空也可以。
